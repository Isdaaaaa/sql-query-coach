# Roadmap

## Phase 1 — Foundations
- Scaffold app structure and design system
- Define data models for queries, schemas, findings, and suggestions
- Prepare sample schemas and queries for demos

## Phase 2 — Heuristic Engine
- Integrate SQL parser and extract tables/joins/filters
- Implement rule set for common anti-patterns and index gaps
- Generate structured findings with severity and rationale

## Phase 3 — Coaching UX
- Build dual-mode UI (Coaching vs Reviewer)
- Render findings, suggested rewrites, and index hints
- Add copy-to-clipboard for rewritten queries

## Phase 4 — AI Commentary Layer
- Add optional AI-generated explanations with clear labeling
- Provide deterministic fallback when AI is disabled

## Phase 5 — Visualization & Demos
- Visualize join shapes and scan risk summaries
- Ship demo flows with canned inputs and screenshots/GIFs

## Phase 6 — Polish & Release
- Harden content, add safety disclaimers, and review wording
- Write README, quickstart, and portfolio highlights
- Record short demo walkthrough
